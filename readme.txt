== Description ==

This plugin allows you to insert high quality YouTube videos and also videos in normal quality.

== Installation ==
1. Upload file mmyYouTubeHQ.php into your WordPress plugin folder
2. Activate the plugin
3. Setup the mmyYouTubeHQ plugin. Go to Settings -> mmyYouTubeHQ


== Usage ==

To embed a YouTube video, first copy the code of the video. E.g. see the url http://www.youtube.com/watch?v=76vBI4gbsw0, the video code is 76vBI4gbsw0.
To embed a YouTube video in <b>normal quality</b>, write <b>[youtube]code[/youtube]</b> into your post or page.
To embed a YouTube video in <b>high quality</b>, write <b>[youtubehq]code[/youtubehq]</b> into your post or page.
If you use the tag <b>[youtubehql]code[/youtubehql]</b> a text is displayed below the video. The text tells the user that this is a high quality video and offers a link to the YouTube website. On the website the user can watch the video in standard quality (because of the internet connectivity). You can edit the text. Go to Settings -> mmyYouTubeHQ
